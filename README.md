# HematoVision: Advanced Blood Cell Classification Using Transfer Learning

## Project Description
HematoVision is a deep learning-based web application that classifies blood cell images into different categories using Transfer Learning and Convolutional Neural Networks (CNN). The system helps in automated blood cell classification which can assist medical professionals in diagnosis.

## Blood Cell Classes
The system classifies blood cells into the following types:
- Eosinophil
- Lymphocyte
- Monocyte
- Neutrophil

## Features
- User Login System
- User Registration System
- Upload Blood Cell Image
- Blood Cell Classification
- Confidence Score Display
- Unknown Image Detection
- Prediction History
- Admin Panel
- Login History
- Register History

## Technologies Used
- Python
- Flask (Web Framework)
- Deep Learning (CNN)
- Transfer Learning (MobileNetV2)
- HTML
- CSS
- JavaScript
- GitHub

## System Architecture
User uploads image → Flask Web Application → CNN Model → Prediction → Result Display → History Stored → Admin Panel

## How to Run the Project
1. Install Python
2. Install required libraries:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000
